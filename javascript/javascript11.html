{

    };
    document.write("<hr>");

    let i =2;
    while(i<=20){
        document.write(i, "<br>");
        i+=2;
    };
    document.write("<hr>");

    document.write("03. for문과 if문<br>");

        if(i%2 == 0){

        };
    };

4줄 : 짝수일 때 document.write()문을 이용해 글색이 빨간색인 짝수 i를 출력하세요
5줄 : i를 2로 나누었을 때 나온 나머지 값이 0이 아닌 다른 값일 때, 짝수가 아닐 때가 조건이다.

10. 구구단 출력하기(5*5 짝수만 나오게 출력하기)
